Web Development

Developing web pages is another one of my passions. Data extracted from sources such as OS files, version control systems, and databases are typically complex and extensive. A very effective way to communicate data to management is to summarize the information on a web page with drill down capability to expand into the details if needed.

In my various roles in the IT field, I have created numerous custom web-based dashboards that were automatically updated with scheduled jobs. This had the advantage of viewing up-to-date information, in a tabular format, and reducing effort in producing manual reports.

Besides the many automated dashboards that have been created for monitoring applications and server hardware parameters, the below web development projects have also been completed:

  • My online profile built with jQuery, CSS, PHP, MySQL, HTML, and AJAX.
  • Insurance Agent's Web Site utilizing AJAX, jQuery, PHP, CSS, Javascript, Photoshop generated graphics, and centralized quote form configuration file for maintainability and scalability
