PROJECT GOALS

The goals of this project are to learn:

  • how to setup an AMP environment for development of web pages on Mac OS X 10.10 Yosemite system
  • how to design a star schema data warehouse
  • how to use Talend to create an ETL transformation
  • how to install Hadoop and setup a data warehouse within HDFS
  • how to create ETL to extract from a traditional OLTP database, transform, and load into a Hadoop data warehouse using Talend's built-in sqoop functionality
  • how to use Hive to extract data from a Hadoop data warehouse
  • how to install and use Tableau to generate dashboards and reports
  • evaluate the benefits of Hadoop, Talend, and Tableau
