MYSQL INSTALLATION

Refer to http://macosxbits.com/2015/03/download-install-mysql-server-in-mac-os-x/ for instructions on how to download and install MySQL. The steps are re-iterated below with some minor changes and additions.

  • Download the MySQL Mac OS X DMG archive from: http://dev.mysql.com/downloads/mysql/
  • Open downloaded file mysql*.dmg, and then open mysql*.pkg file to start installation process.
  • Now follow the steps in installation window and enter your macâ€™s password when prompted.
  • When the installation is completed, you will see something indicating "The installation was completed successfully"
  • Edit your $PATH variable to include the path to the mysql executable. The best way to do this is to create your source file and add the following entry: "export PATH=$PATH:/usr/local/mysql/bin"
  • In the terminal command line, enter "mysql -u root". If you get the "mysql> prompt, then your installation was successful.
  • To integrate MySQL with PHP, do the following:
          cd /var
          mkdir mysql
          cd mysql
          ln -s /tmp/mysql.sock mysql.sock
  • You should assign the root user a password: ALTER USER root IDENTIFIED BY 'new_password'
    Example: ALTER USER 'root'@'localhost' IDENTIFIED BY 'my_new_password'
